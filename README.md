# Prabin KC - Portfolio Site

This repository contains the source code for my personal portfolio, accessible at [PrabinKC.github.io](https://MLucifer13.github.io/PrabinKC.github.io/) (or your custom domain if configured).

## Structure
- `index.html`: The main landing page, built with Tailwind CSS. Showcases projects, skills, and contact information.
- `DESIGN.md`: The color scheme and design tokens utilized behind the dark/light modes.
- `Prabin_KC_Master_CV.docx`: My master resume/CV.

## Features
- AI-Native Builder & Data Analyst portfolio.
- Includes a dark and light mode toggle.
- Hosted statically via GitHub Pages.

## Deployment
Any changes pushed to the `main` branch will automatically be reflected on the GitHub Pages deployment.
